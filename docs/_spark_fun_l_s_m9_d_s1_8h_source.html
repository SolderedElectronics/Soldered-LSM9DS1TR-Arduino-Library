364 // - CTRL_REG1_G = 0x0F: Normal operation mode, all axes enabled.
365 // 95 Hz ODR, 12.5 Hz cutoff frequency.
366 // - CTRL_REG2_G = 0x00: HPF set to normal mode, cutoff frequency
367 // set to 7.2 Hz (depends on ODR).
368 // - CTRL_REG3_G = 0x88: Interrupt enabled on INT_G (set to push-pull and
369 // active high). Data-ready output enabled on DRDY_G.
370 // - CTRL_REG4_G = 0x00: Continuous update mode. Data LSB stored in lower
371 // address. Scale set to 245 DPS. SPI mode set to 4-wire.
372 // - CTRL_REG5_G = 0x00: FIFO disabled. HPF disabled.
